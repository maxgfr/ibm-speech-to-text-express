# ibm-speech-to-text-express

The boilerplate provides an application which uses IBM Speech to Text in an ExpressJS application.

## Configuration

```
    mv .env.example .env # AND SET YOUR OWN CREDENTIALS
    npm install
```

## Launch the application

```
    npm start
```
